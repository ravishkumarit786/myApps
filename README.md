# My Apps

Static web apps published from this repository.

## Mutual Fund Return Calculator

A single-page mutual fund calculator with SIP and lumpsum modes. It includes sliders and numeric inputs for amount, investment period, and expected annual return.

Open `index.html` to view the app locally, or enable GitHub Pages for this repository to publish it on the web.

Published site:

https://ravishkumarit786.github.io/myApps/

## Android / Google Play preparation

The site includes PWA metadata (`manifest.json`), a service worker (`sw.js`), and an install icon (`icon.svg`). These are the first pieces needed to package the website as an Android Trusted Web Activity with Bubblewrap or PWABuilder.
